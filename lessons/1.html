Paint By Numbers

Recently, I’ve started a new activity. It’s called Paint by Numbers. The concept is almost identical to a colouring book that you may have used when you were younger, where there is a key to tell you what colour to use in each section.

The way it works is I’m given a painting. Like the image below. And a series of paint. Rather than having 8 separate tubes of paint lying around, this system has one large connected row of smaller containers, with each container containing a tube of paint. They’re all connected so that we don’t lose any paint. They’re also numbered.



So when I attempt to paint the Sun, I don’t have to search through every single individual loose tube until I find yellow. They are in a structured set of boxes, and I just have to go to the third box. This is really easy. For the smiley face, I just have to go to the 8th box. Then for the heart I go to the 6th box.

In this scenario, the paint set is your array. An array is a data structure.It stores data or information in an ordered way. It makes things easy search, access and find. It can keep similar things together.


New Array called Paints
Paints = 8 Paint Containers
Paints[0] = Blue
Paints[1] = Orange
Paints [2]= Yellow
Paints[3] = Grey
Paints[4] = Green
Paints[5] = Red
Paints[6] = Purple
Paints[7] = Brown

Let’s examine the above. In most cases, our first step is to initialise the array and set the number of containers of paint to 8. This is because arrays are fixed-size, meaning we typically have to define their size before we use them.

Something you may have noticed is that the 8th element in the array is actually put into Paints[7] and not Paints[8]. This is because with the majority of programming languages, indexing starts at 0. This means the first element isn’t paints[1] it’s paints[0].

Alternative worked examples are presented in several languages below:


C
#include <stdio.h>
#include <string.h>
#define NUM_PAINTS 8
#define MAX_SIZE_STRING 100
Int main (){
//initialise empty array
char paints[NUM_PAINTS][MAX_SIZE_STRING];
//add values to array
strcpy(paints[0], "Blue");
strcpy(paints[1], "Orange");
strcpy(paints[2], "Yellow");
strcpy(paints[3], "Grey");
strcpy(paints[4], "Green");
strcpy(paints[5], "Red");
strcpy(paints[6], "Purple");
strcpy(paints[7], "Brown");
//print array
for(int i = 0; i < 8; i++){
printf("%s ", paints[i]);;
}
}

So as a general definition:
An array is a data structure, which can store a fixed-size collection of elements of the same data type.
There are typically three key steps typically in constructing an array

Declare the array name.
Create the array.
Initialize the array values.
